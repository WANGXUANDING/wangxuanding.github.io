2019-5月总结

发表于 2019-05-29 | 0 comments

今天在健身房里想了很多事，恰好又到了月末，干脆写篇本月总结好了

情场

zxx，不知道怎么的，就这么淡了。尝试过挽回，但已经没有什么可以做的了。为什么先撩我的是你，先放手的也是你？原来，只要女生想和你聊天，就不怕没话聊的。从3月初到清明和她去完苏博，一个月多的时间，也算了解了我一桩心事，哪怕大四的时候我没有用那些乱七八糟的PUA套路，我们也不会在一起。

学习

月初开始为之后找工作做打算，看了看面经，原来自己还差一大截。之后，开始撸毕业项目。有一种在青铜局打了很久，突然被拉去打钻石局的感觉。CV这个方向还是神仙打架的那种，感觉自己要惨了。但是，趁着年轻，还没有什么责任包袱的时候试错，总比到三十多岁，房贷，孩子，车贷各种压着，还一心想着转行进退两难好吧？这两天开始在DeepAI上补基础，Udacity教的东西太浅了，到项目的地方难度跨度一下太大。现在不补一些基础，以后要吃亏的。

生活状态

因为zxx那事，一蹶不振了一周。之后想到，她会希望自己男朋友失恋后像我现在这样吗？于是，重回正常生活。月中的时候，把《那些年》再看了一遍，原来我一直以柯景腾为榜样的活着，也不知该说自己是痴情还是幼稚。学生时代的喜欢，又怎么能沿用到现在呢？一副深情的样子表演给谁看？那之后想了两天，决定以后再也不联系了，既然打算以后再也不联系了，那么也就没必要在意自己做的事在她看来幼不幼稚了。微博拉黑，微信拉黑。后来，微信解除黑名单，然后屏蔽她朋友圈了。现在，大概是那种，平时不会联系，但如果哪次去苏州，一定会说“约饭？火锅，我请”这种话的。

失眠的问题又来了，一开始是烦zxx的事，后来就是不想早早的睡觉，再后来开始刷抖音，现在抖音卸了，躺床上一小时依然睡不着。

拿这个表定位我的话，我应该属于中立-混乱。我并不觉得我是好人，只是按着人不犯我我不犯人的原则，你要犯我，我肯定在法律允许范围内，让你吃亏。我好像也从不掩饰自己不是个好人。

人有两种行动准则，

  1. 服从自我
  2. 服从外在准则

大多数人都是属于小时候听父母的，长大后听自己的，我的话，到现在还是属于服从外在准则，可以是法律，规则之类的客体。也不知是哪一步出了错，但是这个需要慢慢来纠正。但是，有谁是100%的1或者2呢。

我这样，能自我激励，比如zxx会希望自己男朋友是这样的之类，坏处好像还挺麻烦，容易被洗脑，容易被情感绑架。

如果还想和zxx在一起，那得加倍努力，让自己配的上她。先不谈女生在年轻时生育价值所带来的杠杆作用以及30岁以后大龄产妇的负收益，至少得不幼稚，不冲动行事，凡事考虑后果，多去接触社会，有稳定工作，工资足够养活两个人，当地有房。不然，让人家姑娘跟你喝西北风啊？这年代，你这岁数还想鼓吹爱情伟大？呵呵。更别说别人不喜欢你了。反正，这套标准，摆哪个女生身上都有用，所以，加油吧。

婚姻必须要有爱情吗？婚姻的基础不是法律吗？要想说爱情伟大，婚姻不需要被法律束缚的人，请同居几年。请问，你们能有孩子吗？房产证上名字怎么办？同居后财产怎么算？到头来，往往是男生占了便宜，女生浪费了青春。

为什么到现在反转证据那么明显了，还有人咬定刘强东qj？？？这种人，我只想甩两个巴掌，甩不到就算了，不和选择性阅读的人生气了。

前两天，阿姐叫我去贴双眼皮贴试试，看看能不能不手术就把双眼皮弄出来。然后顺手看了遮瑕，还有化妆，好麻烦。。。像我这样的懒癌还是算了吧。而且我这眉毛，就不适合走中性风。再加上，我这肤色和易出汗的体质，还是别想着化妆这种事了。

别人伤害了你，你就得做渣男吗？我也没想过做渣男，也做不了，不感兴趣的女生我就不去浪费那时间了，我的相处模式是，认可女生合格了，才会考虑做女朋友。这么一想，我这死宅之后马上要工作了，社交的机会更少了，怎么谈对象啊。总不能真期望着zxx找不到合适的，最后和她在一起吧？她就那样留着个女神的形象挺好的，至少比吵来吵去，还整天被她欺负，最后累了，分手了，像前女友那样好吧。
