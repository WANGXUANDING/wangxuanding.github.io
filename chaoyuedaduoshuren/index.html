如何超越80%的人

发表于 2018-06-01 | 0 comments

超越绝大多数人并不难，只需要以下4点：

  1. 有基本的逻辑分析能力。

  2. 面对自己不了解的事物，在自己初步了解前，不因他人的看法而妄加判断。

  3. 能够长期不断去尝试了解自己不理解的事物。

  4. 认识到即使自己“与众不同”，也并不一定是坏事。

    在123的作用下，你可能会不断地涌现看上去“离经叛道”，但是又有极强逻辑基础的观念。

方法很简单，甚至可以说“弱者”，不过，那绝大多数就是因为太简单了，而时而不见。举例的话，看看社会潜意识对于任何贴有成功学标签事物的厌恶，以及社会大众对于心理学的妖魔化。

如果还是不懂，再举一个微博上的段子，学计算机的=修电脑的，学经济的=管钱的。
